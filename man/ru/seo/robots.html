Модуль представляет собой визуальную оболочку для функционала robots.txt на каждый домен

Для получения содержимого файла robots.txt необходимо выбрать домен и нажать на кнопку "Получить robots.txt". Содержимое файла robots.txt будет загружено в поле "Содержимое robots.txt".

Если в поле "Содержимое robots.txt" нет данных, это значит, что для выбранного домена файл robots.txt строится автоматически по следующему принципу:

User-Agent: Googlebot
Disallow: /?
%disallow_umi_pages%

User-Agent: Yandex
Disallow: /?
%disallow_umi_pages%
Host: $домен
Crawl-delay: 3

User-Agent: *
Disallow: /?
%disallow_umi_pages%

Sitemap: http://{$домен}/sitemap.xml

Макрос %disallow_umi_pages% выводит список запрещенных к индексации страниц (для которых в административной панели была поставлена галочка "Запретить индексацию поисковиками") и следующие строки:

Disallow: /admin
Disallow: /index.php
Disallow: /emarket/addToCompare
Disallow: /emarket/basket
Disallow: /go_out.php
Disallow: /search

Чтобы вернуть файл robots.txt в исходное состояние, нужно удалить все содержимое из поля "Содержимое robots.txt" и нажать на кнопку "Сохранить".
